# 🖼️  Pictures to ASCII-art Converter

**Превращай картинки в текстовый ASCII-арт через удобный GUI**

[![Python](https://img.shields.io/badge/python-3.8%2B-blue)](https://python.org)
[![PyQt5](https://img.shields.io/badge/PyQt5-5.15-green)](https://riverbankcomputing.com/software/pyqt)

---

## 📌 О проекте:

Это десктопное приложение, которое конвертирует изображения в ASCII-графику.  
Вы загружаете картинку (JPG, PNG, BMP), нажимаете кнопку — и получаете текстовое представление, собранное из символов разной плотности.

Интерфейс написан на **PyQt5**.  
Конвертация использует **Pillow** для обработки изображений.

---

## 🚀 Возможности:

- Загрузка изображений через графический интерфейс
- Предпросмотр загруженной картинки
- Автоматическое масштабирование под окно просмотра
